Pinyin
------

a with tones:

-   ā
-   á
-   ǎ
-   à

``` r
a3 <- "&#x1CE;"
i3 <- "&#x1D0;"
hello <- paste0("n",i3," h",a3,"o")
cat(hello)
```

    ## n&#x1D0; h&#x1CE;o

``` r
print(hello)
```

    ## [1] "n&#x1D0; h&#x1CE;o"

In-line output: nǐ hǎo
